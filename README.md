# Mollo Event System
 Out-Off-the-Box Opensource CMS for Event Organiser
 
Basend on Drupal 8 / 9
install via Composer


## Includes
### Events
- 1 Day Events
- multiple Days Event like Operas, Operettes

### Articles
- News, Press, Media

### 
- Reservations, Donations, Simple Ticket System

### Management
- Artists (Solist, Choir, Statists, Orchestra)
- Bands
- Komponists / Writers
- Locations
- Newsletter System
- Donations


## Drupal Modules

### Included Modules:
- Mollo Dashboard
- Mollo Event
- Mollo Artist
- Mollo Member  
- Mollo Donation
- Mollo Newsletter
- Mollo Reservation
- Mollo Ticket
